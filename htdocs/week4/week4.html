3. Differences between a CGI program and a PHP program:
1. CGI is a standard for the communication between the Web server and the external program. So any program that meets with the requirements of CGI can be called as a CGI program, no matter which programming language it uses. In that case, PHP can also be used to create CGI programs, and a PHP program is a specific kind of CGI programs.

2. Since PHP is an embedded language, a PHP program can generate HTML tags more quickly, which is more efficient than a CGI program.

3. PHP process will load the frequently-visited PHP program in the memory. So when the program is accessed again, it can be executed at once without compiling, which accelerates the user’s visiting.

4. PHP supports a wide range of functions while the CGI program is limited by the language it uses.
