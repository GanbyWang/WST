Code Introduction:
  This time, ThinkPHP is used to realize the framework. As other application frameworks, ThinkPHP also follows the MVC model. Base on MVC principle, my code files are divided into three parts as follow:
1. View layer: The view layer of MVC stipulates the appearance of the web application. In this case, submit.html and query.html serve as the templates.
2. Control layer: The control layer of MVC describes the logical flow of the web application. In this case, SubmitController.class.php and QueryController.class.php are responsible for displaying the page and interacting with database.
3. Model layer: The model layer is put forward in order to reuse the code. ThinkPHP offers a lot of models, such as the "M" and "D" functions, which are used in handling database. I use some functions available in ThinkPHP instead of realizing them myself.
